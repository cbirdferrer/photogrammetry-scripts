Instructions

There are two different ways you can use this.

1. Copy all the images you want to run the code on into the folder called "images"
2. Open the .py file in python WIN or in IDLE and run the code from there.

OR

1. Copy the script and the exiftool files into the folder where the image folder is (i.e the Flight folder)
2. Open the .py file in python win or in IDLE and run the code from there.

The tool will then output a .csv file named "metadata_altitudes" in the same folder as the script.
